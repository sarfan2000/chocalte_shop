# Sarfan E-commerce

an E-commerce Website built with MERN stack.

## Instructions

after cloning, run this command in the root folder

```bash
npm install
```

```bash
npm start
```

## Features

- User authentication
- Product search and filtering
- Shopping cart
- Order management
- Admin dashboard

## Technologies Used

- MongoDB
- Express.js
- React.js
- Node.js
